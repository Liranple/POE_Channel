# 빠른 시작 가이드

## 🚀 실행하기

### 1. 개발 서버 시작

```bash
npm run dev
```

### 2. 브라우저에서 확인

```
http://localhost:3000
```

## 📱 사용법

### 탭 전환하기

1. 왼쪽 사이드바에서 원하는 탭 클릭
2. 페이지가 즉시 전환됩니다 (새로고침 없음)

### 플라스크 태그 빌더 사용하기

1. 사이드바에서 "⚗️ 플라스크 태그" 클릭
2. 관리자 모드 ON/OFF 토글
3. 접두어/접미어 옵션 클릭하여 선택
4. 상단 검색 박스에서 결과 확인

### 관리자 모드 기능

- **옵션 편집**: 옵션에 마우스 호버 → ✎ 클릭
- **옵션 삭제**: 옵션에 마우스 호버 → 🗑 길게 클릭
- **새 옵션 추가**: 각 섹션 하단의 + 버튼 클릭

## 🎨 인터페이스 가이드

### 사이드바

- **호버 효과**: 탭에 마우스를 올리면 강조 표시
- **활성 탭**: 파란색으로 표시, 왼쪽에 세로 바
- **아이콘**: 각 탭의 기능을 시각적으로 표현

### 메인 콘텐츠

- **자동 중앙 정렬**: 최대 너비 1100px
- **스크롤 가능**: 내용이 많을 경우 스크롤

### 애니메이션

- **페이지 전환**: 부드러운 페이드인 효과
- **호버**: 버튼과 옵션에 즉각 반응
- **클릭**: 명확한 시각적 피드백

## ⌨️ 키보드 단축키 (향후 추가 예정)

현재는 마우스 클릭으로만 사용 가능합니다.

## 💡 팁

### 플라스크 태그 빌더

1. **여러 옵션 선택**: 클릭으로 여러 옵션 선택 가능
2. **아이템 조건**: 선택한 옵션에 따라 자동 계산
3. **유형 필터**: HP, MP, SP, TK 태그로 필터링

### 최적의 사용 환경

- **브라우저**: Chrome, Edge, Firefox 최신 버전
- **화면**: 1280px 이상 권장
- **모바일**: 반응형 지원

## 🔧 문제 해결

### 페이지가 로드되지 않을 때

```bash
# 1. 개발 서버 중지 (Ctrl+C)
# 2. node_modules 재설치
npm install
# 3. 개발 서버 재시작
npm run dev
```

### 스타일이 적용되지 않을 때

1. 브라우저 캐시 삭제 (Ctrl+Shift+R)
2. 개발 서버 재시작

### 변경사항이 반영되지 않을 때

1. 파일 저장 확인
2. 브라우저 새로고침 (F5)
3. Fast Refresh가 자동으로 적용됩니다

## 🆘 도움이 필요한가요?

### 자주 묻는 질문

**Q: 새로운 탭을 어떻게 추가하나요?**
A: `TABS_GUIDE.md` 파일을 참고하세요. 3단계로 쉽게 추가할 수 있습니다.

**Q: 탭 순서를 바꿀 수 있나요?**
A: `src/components/AppLayout.jsx`의 TABS 배열 순서를 변경하면 됩니다.

**Q: 플라스크 태그 빌더 데이터는 어디서 수정하나요?**
A: `src/components/TagBuilderStash.js` 파일을 수정하세요.

**Q: 색상을 바꾸고 싶어요.**
A: `src/app/globals.css`의 `:root` 변수를 수정하세요.
